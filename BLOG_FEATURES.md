# Blog Management Features

## How to Use the Blog Module

### Creating/Editing Blog Posts

1. **Access the Blog Module**
   - Navigate to Admin Panel → Blog Posts
   - Click "New Post" to create a new blog post

2. **Pasting Content from Word Documents**

   **Method 1: Import Word File (.docx)**
   - Click the "Import from Word" button above the content editor
   - Select your .docx file
   - The content will be automatically converted and inserted into the editor

   **Method 2: Direct Copy-Paste**
   - Copy content from Microsoft Word (Ctrl+C / Cmd+C)
   - Click inside the rich text editor
   - Paste (Ctrl+V / Cmd+V)
   - Formatting, tables, and images will be preserved

3. **Rich Text Editing Features**
   - **Headers**: 6 levels of headings
   - **Text Formatting**: Bold, italic, underline, strikethrough
   - **Colors**: Text and background colors
   - **Lists**: Ordered and bulleted lists
   - **Alignment**: Left, center, right, justify
   - **Links**: Add hyperlinks
   - **Images**: Upload or drag-and-drop images
   - **Tables**: Create new tables (3x3 default) or paste from Word
     - Right-click on table for menu options:
       - Insert/delete rows and columns
       - Merge/unmerge cells
       - Delete entire table
   - **Code Blocks**: Add code snippets
   - **Blockquotes**: Add quoted text

4. **SEO Features**
   - Meta Title: Custom SEO title (defaults to post title)
   - Meta Description: Custom description for search engines
   - Tags: Add multiple tags for categorization
   - Auto-generated slug from title

5. **Publishing Options**
   - Save as Draft: Uncheck "Publish immediately"
   - Publish: Check "Publish immediately"
   - Quick publish/unpublish from the posts table

## Tips

- **Creating Tables**:
  - Click the "Table" button in the toolbar to insert a new 3x3 table
  - Right-click any cell to add/remove rows and columns
  - Select multiple cells and right-click to merge them
- **Pasting Tables**: Copy tables from Word and paste directly - formatting is preserved
- **Table Editing**: Right-click on any table cell to access the context menu for:
  - Adding rows above or below
  - Adding columns left or right
  - Merging selected cells
  - Deleting rows, columns, or the entire table
- **Images**: Can be uploaded, pasted, or drag-and-dropped into the editor
- **Formatting**: All Word formatting is preserved during import
- **Auto-save**: Changes are saved when you click "Create Post" or "Update Post"
